# Simon Game - Full Stack

A full-stack memory game where users can sign up, log in, play the Simon game, and compete on a global leaderboard.

## Live Demo
(After deployment, add your Render link here)

## Features
- User Signup and Login
- Secure password hashing using bcrypt
- Interactive Simon memory game
- Score saving after game over
- Global leaderboard showing highest score per user
- Responsive UI for desktop and mobile

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript (jQuery)

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL
